# Shivin's 1st Birthday Party Invitation 🎂🎈

An interactive, animated cartoon-themed digital birthday invitation card handcrafted with HTML, CSS, and Vanilla JS. It features **Doraemon & Nobita** chibi characters holding the interactive envelope card.

## ✨ Features
* **Interactive Envelope Welcome Page**: A floaty character card with a bouncy manga speech bubble prompting users to open the invitation.
* **Toy Train Background Animation**: A cute toy train chugging along a track at the bottom of the screen with wheels spinning and chimney smoke puffing.
* **Shivin's Polaroid Slideshow**: An interactive 6-photo rotating slide carousel displaying photos of the birthday boy, complete with prev/next navigation controls.
* **Ambient Canvas Particles**: Drifting clouds and floating, spinning Doraemon gadgets (Take-copters and Anywhere Doors).
* **Music Box Synthesizer**: Plays a Web Audio API synthesized "Happy Birthday" melody loop when opened.
* **Interactive Balloons**: Floating parallax balloons that make a pop chime sound and burst into confetti when clicked.
* **Interactive RSVP Form**: Fully integrated form to collect attendance status, names, phone numbers, and wishes.
* **Live Wishes Board**: Renders guest wishes in real-time onto a card board.

## 🚀 How to Deploy on Vercel
This project is built using serverless static assets, meaning it is 100% ready for Vercel out of the box with zero configuration!

1. Go to [Vercel](https://vercel.com) and log in.
2. Click **New Project** and import this GitHub repository.
3. Keep all settings (Build Command, Output Directory) as **Default**.
4. Click **Deploy**! Your invitation will be live in seconds.
